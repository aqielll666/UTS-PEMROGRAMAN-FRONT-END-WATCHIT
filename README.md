Judul Proyek
disini saya membuat aplikasi website bernama WATCHIT, kenapa saya membuat website ini karena saya mendapat kan isnpirasi ketika sedang menonton netflix, dinamakan WATCHIT karena itu langsung muncul saja di kepala saya

Deskripsi Proyek
Aplikasi website ini berbasis HTML, CSS (saya memakai framework tailwind css dan memakai react vite), dan JavaScript untuk bisa membuka tab film mana yang ingin di pilih dan bisa mencari (search film yang di inginkan), untuk database film nya saya memakai API key dari TMDB (The Movie DataBase), itu saya mengikuti nya seperti netflix dia memakai database TMDB juga.
